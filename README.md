
# 🚨 Crime Data Analysis using Python

## 📌 Project Overview

This project focuses on analyzing crime data from Maryland counties (1975–2020) using Python. The aim is to understand crime trends, patterns, and relationships between different types of crimes using data science techniques.

---

## 🎯 Objectives

* Analyze trends of violent and property crimes over time
* Compare crime levels across different counties
* Study the relationship between population and crime
* Perform statistical analysis on crime data
* Visualize crime patterns using graphs

---

## 📂 Dataset

* **Name:** Violent Crime & Property Crime by County (1975 to Present)
* **Format:** CSV
* **Description:** Contains year-wise crime data including population, violent crimes, property crimes, and crime rates.

---

## 🛠 Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔍 Process Followed

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Statistical Analysis

---

## 📊 Visualizations

* Line Chart (Crime Trend Analysis)
* Bar Chart (County-wise Comparison)
* Histogram (Crime Distribution)
* Scatter Plot (Population vs Crime)
* Heatmap (Correlation Analysis)
* Box Plot (Murder Rate Distribution)
* Stacked Bar Chart (Crime by Decade)
* Pairplot (Multi-variable Analysis)

---

## 📈 Key Insights

* Property crimes are higher than violent crimes
* Crime trends vary across different decades
* Population has a direct impact on crime rates
* Strong correlation exists between different crime types
